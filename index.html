        Gary Innerarity · MBA · Solutions Engineer

        About Gary Innerarity

        Gary Innerarity is a Solutions Engineer holding an MBA, based in the United States. He specializes in enterprise agentic AI systems, LLM orchestration, digital thread platforms, digital twin simulations, multi-cloud infrastructure architecture, and platform engineering. He holds more than 14 professional certifications issued by Amazon Web Services, Cisco, CompTIA, Google, Databricks, and Microsoft.

        Agentic AI and LLM Orchestration

        Gary designs and builds production-grade agentic workflows that orchestrate multi-step LLM pipelines, custom tools, and specialized agents that reason, retrieve, and act across live enterprise systems. He has built a multi-tenant, multi-provider agentic coding assistant — called Agentic Coder — that integrates with enterprise SKILLS repositories to help engineering teams write, review, and optimize code across languages and frameworks. He develops composable building blocks for agentic systems including purpose-built agents with defined roles, reusable skills, custom tool and function calling integrations, and Model Context Protocol (MCP) servers. His RAG pipeline implementations index enterprise knowledge bases for real-time retrieval by LLM agents.

        Digital Thread Platform

        Gary designed and built a full-stack enterprise digital thread platform at Vertasyn connecting metadata, artifacts, and cross-system relationships into a continuous data thread. The platform enables end-to-end traceability across the engineering lifecycle, giving decision-makers a unified view of how requirements, designs, systems, and outcomes relate across programs. The frontend is built in React; the backend exposes REST APIs integrating with existing enterprise data sources.

        Digital Twin Simulations

        Gary builds digital twin simulations — virtual replicas of physical systems — that feed real-time operational data back into enterprise decision loops. These systems enable predictive analysis, scenario modeling, and outcome simulation before committing real-world resources, reducing risk and accelerating engineering decisions.

        Automated Cyber Compliance

        Gary designed and built continuous automated compliance pipelines integrated into the Vertasyn platform, spanning from policy-as-code authoring through CI/CD enforcement to real-time audit dashboards. These pipelines eliminate manual compliance checks and provide continuous, auditable evidence of control adherence for cloud environments.

        Infrastructure and Cloud Architecture

        Gary engineered a reusable Terraform module library covering multi-cloud deployments across Amazon Web Services and Google Cloud Platform, standardizing IaC patterns across engineering teams to eliminate configuration drift and accelerate provisioning. His cloud certifications include AWS Certified Solutions Architect Associate and AWS Certified Cloud Practitioner, both issued by Amazon Web Services. He operates a personal homelab running Proxmox, Kubernetes (K3s clusters), Helm, and Docker to develop and test infrastructure patterns before enterprise deployment.

        Monte Carlo Simulation and Risk Analysis

        Gary developed probabilistic risk analysis tooling in Python using Monte Carlo simulation to support decision-making for complex engineering and cost estimation scenarios, enabling teams to quantify uncertainty and model outcome distributions before committing to project plans.

        Certifications

        Gary holds 14 or more professional certifications: AWS Certified Solutions Architect Associate (Amazon Web Services), AWS Certified Cloud Practitioner (Amazon Web Services), AWS Partner Accreditation Technical (Amazon Web Services), AWS Partner Security Best Practices Technical (Amazon Web Services), Cisco CCNA (Cisco), Cisco CCENT (Cisco), Cisco Meraki Network Operator (Cisco), CompTIA Security+ (CompTIA), Databricks Generative AI Fundamentals (Databricks), Google Cloud Platform Fundamentals (Google), Microsoft Technology Associate (Microsoft), AMP Admin I, AMP Admin II, and AMP Admin Security.

        Technical Stack

        Cloud platforms: Amazon Web Services, Google Cloud Platform, Microsoft Azure. Container orchestration: Kubernetes, K3s, Helm, Docker, Proxmox. Infrastructure as Code: Terraform, Ansible, AWS CloudFormation. Languages: Python, JavaScript, Bash, SQL, PowerShell. AI and ML: agentic workflows, LLM orchestration, RAG pipelines, MCP, multi-agent systems, Databricks, generative AI. Networking: Cisco, Meraki, VPC design, SD-WAN.
